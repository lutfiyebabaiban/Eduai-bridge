# Kullanıcı Akış Diyagramı (User Flow)

1. **Giriş:** Kullanıcı uygulamayı açar (Karşılama Ekranı).
2. **Profil Oluşturma:** Kullanıcı hedeflediği kariyer yolunu (Örn: Flutter Developer) ve mevcut seviyesini seçer.
3. **AI Analiz Süreci:** "Yol Haritamı Oluştur" butonuna basıldığında Gemini API üzerinden veri işlenir.
4. **Roadmap Görünümü:** Kullanıcıya özel hazırlanan modüller (Örn: Dart Temelleri -> Widgetlar -> State Management) listelenir.
5. **Detay ve Kaynak:** Her bir modülün üzerine tıklanarak AI tarafından önerilen çalışma kaynakları görüntülenir.
6. **Tamamlama:** Kullanıcı tamamladığı adımları işaretleyerek ilerlemesini kaydeder.
