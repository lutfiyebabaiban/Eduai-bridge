# Teknoloji Yığını (Tech Stack)

### Frontend
- **Framework:** Flutter (Mobile & Web)
- **Dil:** Dart
- **State Management:** Provider (Hafif ve etkili veri yönetimi için)

### Backend & AI
- **AI Model:** Google Gemini API (Generative AI)
- **Database/Auth:** Firebase (Kullanıcı verilerini ve ilerlemeyi senkronize etmek için)

### Araçlar
- **Versiyon Kontrol:** Git & GitHub
- **Deployment:** Firebase Hosting (Web yayını için)
